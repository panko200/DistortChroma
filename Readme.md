## 概要

YukkuriMovieMaker4 にて動作する映像エフェクトプラグインです。  
色ズレありVectorBlurらしいです。

## 使用方法

1. プラグインをインストールして YMM4 を起動する。
2. アイテムを追加し、映像エフェクトから、「ディストートクロマ」を適用する。
3. 任意の設定にすれば、動きます。

## アンインストール方法

1. YMM4 を起動して `ヘルプ(H)` > `その他` > `プラグインフォルダを開く` をクリックする。
2. YMM4 を終了する。
3. `DistortChroma` という名前のフォルダを削除する。

## 注意点

OS : Windows11 (64bit)  
ゆっくりMovieMaker4 : v4.51.0.3  
にて動作確認をしています。
